ManagedObjectWrappingModel

public protocol ManagedObjectWrappingModel : Equatable

A protocol to mark Models so that they can be used by a managed object which conforms to ManagedObjectModelWrapping to automatically wrap them. A model should conform to this protocol to act as a facade for the wrapped managed object. The idea is to provide a better and type-safe interface with a model to the managed object.
